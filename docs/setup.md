### MacOS

#### Brew

- Download brew
- Add brew to path (follow prompts)

#### Iterm2

- install iterm2
- sync iterm settings
- set profile as default

#### CLI packages

- install dev packages - `install_packages()`
- gh auth login

#### Dotfiles

- clone dotfiles

#### Shell

- symlink `~/.zprofile`
- symlink `~/.zshrc`
- install oh my zsh
- install zsh-autosuggestions
- install zsh-syntax-highlighting
- symlink ohmyzsh alias

#### Node

- install nvm
- `nvm install node`
- install yarn

#### Ruby

- install rbenv
- gem update bundler
- gem install rails

#### DB stuff - if required

- install sqlite and postgres

#### VS Code

- set vscode path - via vscode command palette
- sign in to vscode

#### Slack

- install slack - brew
