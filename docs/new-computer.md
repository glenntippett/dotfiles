### MacOS

- Download brew
- Add brew to path (follow prompts)

- install iterm2
- sync iterm settings
- set profile as default

- install dev packages - `install_packages()`
- gh auth login
- clone dotfiles

- symlink `~/.zprofile`
- symlink `~/.zshrc`
- install oh my zsh
- install zsh-autosuggestions
- install zsh-syntax-highlighting
- symlink ohmyzsh alias

- install nvm
- `nvm install node`
- install yarn

- install rbenv
- gem update bundler
- gem install rails

- install sqlite and postgres

- set vscode path - via vscode command palette
- sign in to vscode

- install slack - brew
